# Hamsa Reminders Web

React frontend for the Hamsa Reminders Service.

## Tech Stack

- React
- TypeScript
- Vite
- Axios
- Socket.IO Client
- CSS

## Features

- Create project reminders
- List user reminders
- Edit pending reminders
- Cancel reminders
- Realtime reminder fired notifications
- Sound notification
- Browser notification
- Toast messages
- Fired reminder overlay
- Hamsa-inspired UI style
